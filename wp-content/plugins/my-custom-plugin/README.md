# My Custom Plugin

## Overview
This is a custom WordPress plugin designed to enhance the functionality of your personal blog. It provides full CRUD (Create, Read, Update, Delete) capabilities for managing blog content.

## Features
- Custom post types
- Shortcodes for easy content embedding
- User-friendly interface for managing posts

## Installation

1. **Download the Plugin**
   - Download the `my-custom-plugin` folder from the repository.

2. **Upload to WordPress**
   - Navigate to your WordPress installation directory.
   - Go to `wp-content/plugins/`.
   - Upload the `my-custom-plugin` folder here.

3. **Activate the Plugin**
   - Log in to your WordPress admin dashboard.
   - Go to `Plugins` > `Installed Plugins`.
   - Find `My Custom Plugin` and click `Activate`.

## Usage

- After activation, you will find a new menu item in the WordPress admin sidebar for managing your custom posts.
- Use the provided interface to create, read, update, and delete your blog posts.

## Uninstallation

To uninstall the plugin:

1. Deactivate the plugin from the `Plugins` menu.
2. Optionally, delete the plugin folder from `wp-content/plugins/`.

## Troubleshooting

- If you encounter issues, ensure that your WordPress installation is up to date.
- Check the plugin's compatibility with your current theme and other plugins.
